# Notes Manager

A full-stack notes management application built with Node.js and Express.

## Live Demo
https://notes-manager-nic8.onrender.com

## Features
- Create notes with title and content
- View all saved notes
- Delete notes
- Persistent storage
- Responsive design

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Storage:** JSON file
- **Deployment:** Render

## Run Locally
git clone https://github.com/NamrataSalgar427/notes-manager.git
cd notes-manager
npm install
node server.js

Visit http://localhost:3000
